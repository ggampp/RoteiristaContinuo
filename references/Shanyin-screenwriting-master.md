# Shanyin-screenwriting-master

**Link:** https://github.com/Shanyin-ai/shanyin-screenwriting-master

.skill file pronta para importar em Claude, GPT e ferramentas compatíveis.

Focada especialmente em formatos curtos:
- **1-3 minutos**: Concept ultra-shorts (how-to-tell / what-if)
- **5-10 minutos**: Narrative shorts com arco completo

**Workflow da skill:**
1. Design de personagens (com arcos e traços contrastantes)
2. Estrutura narrativa (3 atos ou Hero’s Journey)
3. Quebra de cenas com linguagem de câmera
4. Roteiro completo em formato padrão de cinema

Ótima base para criar skills de roteirista e entender como estruturar prompts para conteúdo curto.
