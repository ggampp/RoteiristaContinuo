# ArcReel

**Link:** https://github.com/ArcReel/ArcReel

Workbench open-source (AGPL) totalmente dirigido por AI Agents (Claude Agent SDK + Skills + Subagents).

Fluxo completo: Novel → Extração de personagens/cenários → Roteiro JSON → Design de personagens (imagens de referência) → Storyboard → Vídeo.

**Melhor em:** Continuidade visual forte através de imagens de design de personagens + "Clue Tracking" para props e elementos de cena.

Suporta muitos modelos (Veo 3.1, Seedance, Grok Video, Sora, Kling etc.) e exporta para CapCut.

Altamente recomendado como backend completo para pipeline de vídeo.
