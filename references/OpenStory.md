# OpenStory

**Link:** https://openstory.so/ + https://github.com/openstory-so/openstory

Plataforma open-source (MIT) que transforma um script ou uma única ideia em um vídeo multi-cena completo, com consistência automática.

**Como funciona:**
- Você define personagens, locações e estilo visual **uma única vez**.
- O sistema mantém essa identidade visual em **todos os shots** da sequência.
- Gera imagens + motion video + trilha sonora.

**Melhor em:** Produções narrativas curtas com identidade visual unificada (trailers, vídeos de história, conteúdo de marca).

Excelente para quem quer consistência sem precisar gerenciar referências manualmente em cada plano.
