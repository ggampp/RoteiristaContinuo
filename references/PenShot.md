# PenShot (story-shot-agent)

**Link:** https://github.com/neopen/story-shot-agent

Agente baseado em LangGraph que pega qualquer roteiro e o decompõe em prompts otimizados para modelos de vídeo (Sora, Veo, Runway, Kling), mantendo consistência narrativa.

**Melhor em:** Continuidade via multi-level memory + Chroma vector retrieval. Suporte nativo a **MCP**.

Perfeito para integrar como ferramenta no seu harness de agentes (especialmente se você usa MCP).
